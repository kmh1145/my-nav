# ✨ 毛玻璃导航仪表盘

一个具有毛玻璃（Glassmorphism）风格的个人常用网站导航页，支持自定义背景、数据导入导出等功能。

## 🎯 特性

### 核心功能
- 🪟 **毛玻璃磨砂效果** — 现代感十足的 UI 设计
- 🎯 **仪表盘风格** — 自由拖拽排序
- 🔍 **搜索过滤** — 快速找到需要的网站
- 🌙/☀️ **主题切换** — 暗色/亮色主题一键切换
- ⏰ **实时时钟** — 显示当前时间
- 🌤 **天气显示** — 自动获取天气信息
- ⭐ **粒子星空** — 动态粒子背景效果
- 📤 **自定义图标** — 支持上传网站图标
- ⌨️ **键盘快捷键** — 提高操作效率
- 💾 **本地持久化** — 数据保存在浏览器
- 📱 **响应式设计** — 完美适配各种设备

### 🆕 新增功能
- 🖼️ **自定义背景图** — 支持上传本地图片或输入 URL
- 📦 **数据导入导出** — JSON 格式备份和恢复
- ⚙️ **设置面板** — 集中管理所有配置
- 📁 **默认配置** — 首次访问自动加载预设配置

## 🚀 部署

### GitHub Pages（推荐）

1. **Fork 或创建仓库**
   - 将项目上传到你的 GitHub 仓库

2. **启用 GitHub Pages**
   - 进入仓库 → Settings → Pages
   - Source 选择 `main` 分支
   - 保存

3. **访问网站**
   - `https://<username>.github.io/<repo>/`

### Vercel

1. 导入 Git 仓库
2. 无需配置，直接部署
3. 自动获得 HTTPS 域名

### Cloudflare Pages

1. 连接 GitHub 仓库
2. 构建设置保持默认
3. 部署完成

### 本地使用

直接在浏览器中打开 `index.html` 即可。

## 🎨 自定义

### 修改默认配置

1. **导出当前配置**
   - 在网站中点击 ⚙️ 打开设置
   - 点击「📤 导出数据」
   - 下载 JSON 文件

2. **替换默认配置**
   - 将导出的 JSON 内容替换 `default-config.json`
   - 提交并推送到 GitHub

3. **其他设备自动加载**
   - 新设备首次访问时会自动加载 `default-config.json`
   - 之后的修改保存在浏览器本地

### 自定义背景图

1. 点击 ⚙️ 打开设置
2. 在「自定义背景」区域：
   - 点击「上传图片」选择本地图片
   - 或输入图片 URL 后点击「应用」
3. 选择背景效果：
   - **覆盖 (Cover)** — 图片铺满整个背景
   - **适应 (Contain)** — 图片完整显示
   - **平铺 (Repeat)** — 图片重复排列
4. 调节遮罩透明度（0-100%）
5. 点击「保存所有设置」

### 数据备份与恢复

**导出数据：**
1. 点击 ⚙️ 打开设置
2. 点击「📤 导出数据」
3. 下载 JSON 备份文件

**导入数据：**
1. 点击 ⚙️ 打开设置
2. 点击「📥 导入数据」
3. 选择之前导出的 JSON 文件
4. 确认覆盖

## 🔧 高级配置

### 修改默认网站列表

编辑 `script.js` 中的 `DEFAULT_DATA` 对象：

```javascript
const DEFAULT_DATA = {
  categories: [
    {
      id: 'cat_default_1',
      name: '我的网站',
      emoji: '⭐',
      websites: [
        {
          id: 'web_default_1',
          name: '网站名称',
          url: 'https://example.com',
          desc: '网站描述',
          icon: null
        }
        // 添加更多网站...
      ]
    }
    // 添加更多分类...
  ],
  settings: {
    theme: 'dark',  // 默认主题：dark 或 light
    categoryOrder: [],
    websiteOrder: {}
  }
};
```

### 修改主题颜色

编辑 `style.css` 中的 CSS 变量：

```css
:root {
  /* 亮色主题 */
  --accent: #6366f1;           /* 主色调 */
  --accent-hover: #818cf8;     /* 悬停颜色 */
  --accent-glow: rgba(99, 102, 241, 0.3);  /* 发光效果 */
}

[data-theme="dark"] {
  /* 暗色主题 */
  --accent: #818cf8;
  --accent-hover: #a5b4fc;
  --accent-glow: rgba(129, 140, 248, 0.3);
}
```
